Super Hyper Space

Super Hyper Space is a neo-classical top-down shooter, taking place in a futuristic space setting.

With Super Hyper Space I wanted to create a good balanced and simple shoot ‘em up game with an upgrade Mechanic that everybody can enjoy. My goal was to develop a game from start to finish by myself. This includes not only the programming but also the game design and creation of graphics. During the development I learned a lot about game developing and Player Feedback helped me to improve the game even further.

Upgrade Mechanic

The game has an upgrade mechanism. You can buy new weapons, upgrade your shield or buy addons, which for example attract coins from further away or show the life of the enemy. To do this, the player needs more energy, which is supplied by an energy core.

To upgrade you need to collect coins, you get for destroying enemies. While stronger opponents lose more coins, stronger upgrades cost more coins.

While the main game loop is about collecting more coins to buy better upgrades and defeat stronger opponents, it is also about organizing the energy costs. To this end, players can choose which weapons and upgrades they want to upgrade For some missions, a faster ship with a large radius weapon is a good choice. For other missions, a better shield and a weapon with more penetrating power is an advantage. The game gives each player the opportunity to play in their own style.

In order to organize the energy costs, the game offers a preview window where the player can see how much energy the current devices are using. Below this is a window that lists the costs in detail, so that the player has a good overview of all investments.

Lost Legend

Lost Legend is a retro 2D action role-playing game with puzzle elements in a fantasy scenario.

This game was inspired by the 16-bit Super Nintendo games like Secret of Mana, Legend of Zelda and many others. As much as I love these old games, it has always been my dream to redesign such a game with fast and smooth gameplay in a modern look. I developed this project completely on my own. This includes the programming, the graphics and the game design.

Lock and Key Level Design

Watching the web series Bosskeys of the Youtube channel GameMakers Toolkit I learned a lot about non-linear level design in games like Zelda, Dark Souls, and Metroid. This helped me a lot in creating my level design. I took over the graph system from the creator Mark Brown for creating my own level graphs. Any smallkey is used to open any locked door . Each key object can be used to overcome many obstacles . And every boss key can be used to open exactly one boss door . The example below shows a relatively simple layout for one level. But with the system more complex levels are possible. Since this system has helped me so much, I have developed it further, as you can see in my next project.

Dependency Level Generator

The Dependency Level Generator is a project that aims to generate prozudenural level sections or entire games

The generator creates different types of dependencies and resolves them so that a solution is always possible. Randomness and various configurable options have an influence on the result. Possible are games like action adventure such as Legend of Zelda, survival horror games like the classic Resident Evil games, but also other genres like Point-and-click adventure games or physical pen-and-paper are thinkable.

Here is a very simplified description of how it works.

1. The start room is created.
2. A main branch is created. This branch can be empty, contain a mini boss, a key item or other. It is random which room contains the item.
3. Next a locked room with a higher key level is created.
4. A sub branch is created at a random place. If the generator has already created the key item, it can also be locked with the key item.
5. Points 2 to 4 are repeated until all branches are created. The last point always contains the boss room and is locked with the boss key.
